## 開発目的
- 正木研究室での、後期期間を通したVRゲームのチーム制作

## 開発期間
- 2022年10月6日～2023年1月12日  
## 開発環境
- C#
- Unity2021.3.0f  
- VisualStudio2019  
- GitHub  
- Visual Studio Code  
- Meta Quest 2  
## チーム構成
- 企画担当3人、デザイン担当2人、プログラム担当4人です。
